# Crud App

A crud app made with Vue and Flask.

# Installation

## API
navigate to api folder
```
cd api
```

install dependencies
```
pip install -r requirements.txt
```

start server
```
python3 run.py
```

## View
navigate to view folder
```
cd web
```

install dependencies
```
npm install
```

serve
```
npm run server
```

# Usage

View on http://localhost:8080
