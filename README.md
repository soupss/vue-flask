# Crud App

A crud app made with Vue and Flask.

# Installation

## API
```
cd api
```

#### install dependencies
```
pip install -r requirements.txt
```

#### start server
```
python3 run.py
```

## View
```
cd web
```

#### install dependencies
```
npm install
```

#### serve with hot reload at localhost:8080
```
npm run server
```

# Usage

View on http://localhost:8080
